# 客户地址处理与 Python 入门

公司拥有非常广泛的客户群，需要了解客户集中在哪些区域。为了解决这个任务，他们会从数据库中获取所有客户地址，每个地址看起来都像这样：

```
address = "29 E Madison St., Chicago, IL, 60602"
```

为了按区域对客户进行分组，我们不需要完整的街道地址。我们只需要邮政编码。我们知道邮政编码始终是地址末尾的五个字符。我们可以使用`-5`作为起始索引来切片：

```
address[-5 : : ]
```

为了练习，尝试从这个地址中切分出城市和州。请记住，在 Python 中空格也算作一个字符，并且会有索引。
