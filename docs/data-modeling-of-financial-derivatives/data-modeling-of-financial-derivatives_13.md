# 在交付日当天，农夫 B 意识到争议中的奶牛实际上已经怀孕，因此拒绝交付，除非农夫 A 支付更多费用。农夫 A 拒绝并提起诉讼。尽管这是一起相对直接的违约案件，但诉讼结果可能并不那么确定。例如，如果双方在合同中均认同了奶牛无法生育这一错误的重要事实，法院可能会考虑这一事实并支持农夫 B 的立场。³

合同可能违反某种类型的`REGULATION`（可细分为`STATE REGULATION`或`FEDERAL REGULATION`）。例如，投资者 A 过去可能违反了联邦法律，并被禁止在`T2`时间点之前签订任何类型的金融合同。如果该投资者在`T2`之前签订了金融合同，该合同可能会被联邦监管机构宣告无效。你可能需要对这些情形进行建模（图 3-15）。⁴
