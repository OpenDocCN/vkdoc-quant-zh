# 前言

*永远努力征服自己，而非命运。*

——勒内·笛卡尔，《方法论》

最近我目睹了两位好友间的争论，两人均为数据建模从业者。争论焦点在于数据建模是否是一门艺术形式。一方认为成功的数据建模师是软件专业人士中精挑细选的特殊群体，其技能无法传授；要么天生具备，要么无缘此道。另一方则认为几乎任何人都能学习数据建模并成为成功的从业者。

我未参与讨论，宁愿置身事外。作为戴尔·卡耐基的忠实学生，我知道“赢得朋友并影响他人”的最佳方式就是完全避免这类争论。但现在，我借此机会敞开心扉表达对此主题的看法。我不认为数据建模是与生俱来的艺术。对我而言，数据建模是一种习得的技能，会随着经验积累不断提升。像任何其他职业一样，数据建模需要投入、练习和热情。大多数知识都会随时间流逝而消亡，除非不断更新和运用。要精通并保持某项技能，你需要刻苦练习。正是通过练习和行动，你才能真正学习并对自己所做的事充满信心。
